# lorem-ipsum-generator

A lorem ipsum placeholder text generator built with HTML, CSS, and JavaScript.

## Features

- [x] Input field to specify number of paragraphs (1–20)
- [x] Generated paragraphs displayed with copy button per paragraph
- [x] Copy All button to copy all paragraphs at once
- [x] First paragraph always starts with classic lorem ipsum opening
- [x] Auto-generates on page load

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)

## How to run

Open `index.html` in your browser — no build tools needed.
